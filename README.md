# Social Media Mockup Generator

Erstellt realistische Social-Media-Mockups für Präsentationen und Dokumente.

## Unterstützte Formate

- **Instagram**: Post (1:1), Story (9:16), Reel (9:16)
- **TikTok**: Feed-Video (9:16)

## Setup

```bash
npm install
npm run dev
```

Dann im Browser öffnen: http://localhost:3000

## Deployment auf Vercel

1. Repository auf GitHub erstellen und Code pushen
2. Auf [vercel.com](https://vercel.com) einloggen
3. "Import Project" → GitHub-Repository auswählen
4. Deploy klicken – fertig!

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- react-easy-crop
